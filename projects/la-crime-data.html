LA Crime Data Analysis (2020–2023)

Comprehensive analysis of crime patterns in Los Angeles using geospatial visualization

GitHub Repository

Overview

This project analyzes crime data from Los Angeles between 2020 and 2023, identifying patterns across different areas, weapon types, and time periods. Using advanced data visualization techniques, including heatmaps and interactive plots, the analysis provides insights into crime trends that could help inform public safety strategies and resource allocation.

Problem Statement

Understanding crime patterns in large metropolitan areas is crucial for effective law enforcement and public safety planning. However, raw crime data is often overwhelming and difficult to interpret. This project aimed to transform complex crime statistics into actionable insights by identifying spatial and temporal patterns, weapon usage trends, and demographic correlations.

Approach & Tech Stack

  • Data Source: Los Angeles Police Department's open data portal
  • Data Processing: Python with Pandas for cleaning and transformation
  • Visualization: Matplotlib for static plots, Folium for interactive maps
  • Geospatial Analysis: GeoPandas for spatial data processing
  • Statistical Analysis: SciPy for correlation and trend analysis

Key Features

  • Interactive crime heatmaps showing concentration across LA neighborhoods
  • Time-series analysis of crime rates by hour, day, month, and year
  • Weapon type distribution analysis across different crime categories
  • Demographic correlation studies between crime rates and socioeconomic factors
  • Comparative analysis of pre and post-pandemic crime patterns

Architecture

The project follows a data pipeline architecture starting with data acquisition and cleaning, followed by exploratory data analysis, statistical modeling, and visualization. The analysis is structured in modular Jupyter notebooks, with each focusing on a specific aspect of crime analysis (spatial, temporal, categorical, etc.). Final visualizations are exported as interactive HTML files for easy sharing with stakeholders.

Challenges Faced

  • Handling missing or inconsistent location data in crime reports
  • Creating meaningful visualizations that balance detail with clarity
  • Accounting for reporting biases in different neighborhoods
  • Optimizing performance for large geospatial datasets

Future Improvements

  • Implement predictive modeling to forecast crime hotspots
  • Integrate additional datasets like economic indicators and population density
  • Develop an interactive dashboard for real-time crime analysis
  • Expand analysis to include comparative studies with other major cities
